Aprendi o que é tipográfia

Aprendi no capitulo 14 também, o que é a tipográfia. Aprendi que as letras/Glífos tem nomes diferentes e características unicas, que as diferenciam, algo interessante é que os glífos tem seu meio de medida proprío que é o EM uma medida que usa a base do "M" maíusculo como base de medida para o restante dos glífos. Também tem a outra medida para os glígos que é o EX que usa como referencia o "x" Minúsculo
